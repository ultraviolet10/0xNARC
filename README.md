
# 0xNARC

Add these to your `.env.local`

    OPENAI_API_KEY=
    DEPLOYER_WALLET_MNEMONIC=

The contracts are located in the /contracts folder.
Deploy to respective chains using Remix, make sure the NFT_MINTER is the owner of the NFT contract.
